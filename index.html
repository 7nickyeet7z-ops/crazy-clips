Crazy Clips

Crazy Clips — private tool to generate short clips and upload drafts to TikTok Inbox for manual posting.

What this app does

  • Uses a long-format video created by user and generates short vertical clips.
  • Creates subtitles and a caption suggestion for each clip.
  • Uploads each clip as a draft to the user’s TikTok Inbox using TikTok API.
  • User manually reviews/edits and posts inside TikTok.

TikTok API Products & Scopes used

  • Login Kit: authenticate the user.
  • Content Posting (Draft/Inbox Upload): uploads videos to Inbox (no auto-publish).
  • Scopes: user.info.basic, video.upload (draft upload)

Data handling

  • Tokens are stored locally on the user’s machine.
  • Videos are processed locally and then uploaded to TikTok Inbox.
  • After successful upload, files can be archived/deleted by the user’s settings.
